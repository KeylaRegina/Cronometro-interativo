# ⏱️ Cronômetro Interativo com Ampulheta

Projeto desenvolvido durante meus estudos em desenvolvimento web, com o objetivo inicial de criar um cronômetro simples.

Além dos requisitos propostos, implementei uma **ampulheta interativa**, tornando a experiência mais visual e dinâmica.

---

## 🚀 Funcionalidades

* ▶️ Iniciar o cronômetro
* ⏸️ Pausar o tempo
* 🔄 Resetar o cronômetro
* ⌛ Ampulheta interativa 
* 💻 Interface simples e intuitiva

---

## 💡 Diferencial do Projeto

O desafio proposto no curso era criar um cronômetro básico.
Como forma de praticar e ir além, desenvolvi uma **ampulheta animada**, trazendo um elemento visual que representa a passagem do tempo.

Esse incremento demonstra minha capacidade de:

* Pensar além do solicitado
* Melhorar a experiência do usuário (UX)
* Aplicar criatividade em projetos simples

---
